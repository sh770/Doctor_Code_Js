ג'אווה סקריפט המדריך המלא 2023
מערכים תרגיל מספר 30 חיפוש והחלפה

בתרגיל ג'אווה סקריפט הבא נלמד איך לקבל מערך של מספרים, ערך החלפה וערך חיפוש, נחפש בתוך המערך ונחליף כל מאורע שנמצא עם ערך ההחלפה

  • a. declare 3 variables numArray, searchTerm and replaceWith.
  • b. write a function searchAndReplace(), that search numArray for searchTerm, and replace it with replaceWith content.
  • b. first use for or while loops then learn to use array.forEach(). 

let numArray = [0, 0, 0, 2, 2, 3, 1, 2, 3, 1, 0]
let searchTerm = 0
let replaceWith = 'zero'

const searchAndReplace = (numArray, searchTerm, replaceWith) => {
    for (let i = 0; i < numArray.length; i++) {
        if (numArray[i] === searchTerm) {
            numArray[i] = replaceWith
        }
    }
}

searchAndReplace(numArray, searchTerm, replaceWith)
console.log('array result', numArray)

numArray.forEach((el, i) => el === searchTerm && (numArray[i] = replaceWith))
console.log('array result', numArray)
