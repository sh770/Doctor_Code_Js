בתרגיל ג'אווה סקריפט הבא נלמד איך לקבל מערך של מספרים, נרשום פונקציה אשר תפקידה לקבל את המערך שקלטנו ולהזיז איברים ממיקום א' למיקום ב'

  • a. create this array [10, 20, 30, 50, 60, 90] and save it as numArray.
  • b. write a function move() that expect 2 parameters from and to.
  • c. switch between the elements, e.g , move(0,3) will result with [50,20,30,10,60,90].
  • b. first use for or while loops then learn to use array.splice(). 

let numArray = [10, 20, 30, 50, 60, 90]
const move = (from, to) => {
    let temp = numArray[from]
    numArray[from] = numArray[to]
    numArray[to] = temp
}
move(0, 3)
console.log('my array after movement is:', numArray)
